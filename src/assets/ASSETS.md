# Asset Manifest — Vault Landing

## Текущие проблемы
- Все иконки 1024x1024 (1.5-3 MB) при отображении 20-40px — перерасход в 50-150 раз
- Нет WebP/AVIF оптимизации
- Имена файлов нечитаемые (хеши figma)

## Рекомендованная структура

```
src/assets/
├── icons/          ← маленькие иконки (табы, кнопки)
├── stickers/       ← персонажи и декор для hero-секции
├── brand/          ← логотипы, App Store бейджи
└── og/             ← Open Graph изображения для соцсетей
```

## Рекомендации по генерации

- **Стиль:** comic/cartoon, толстые контуры (2-4px), яркие насыщенные цвета
- **Палитра:** оранжевый (#E95600, #FDAC04), белый, чёрные контуры
- **Прозрачность:** все кроме OG-картинок — PNG с прозрачным фоном
- **Формат:** PNG-24 для прозрачных, можно JPEG для OG
- **Стиль стикеров:** как Telegram-стикеры — чёткие контуры, мультяшные, с лёгкой тенью
